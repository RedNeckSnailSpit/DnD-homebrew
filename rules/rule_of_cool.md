# Rule of Cool - Homebrew Rule

## Core Principle
If a player attempts something creative, cinematic, or genuinely cool that isn't explicitly covered by the rules, the DM may allow it with appropriate checks and consequences.

## When to Apply
- Player describes an action that's narratively awesome
- The action makes sense in context
- It doesn't break game balance
- It enhances the story or moment

## How to Apply
1. **Determine if it's physically/magically possible** in the world
2. **Assign an appropriate check** (often higher DC than standard)
3. **Consider consequences** - success AND failure outcomes
4. **Make it memorable** - describe the result cinematically

## Examples
- Using an enemy as a springboard to reach a higher ledge (Athletics check)
- Sliding under a closing portcullis while firing a crossbow (Acrobatics + attack roll)
- Channeling a spell through a metal object to create an improvised effect (Arcana check)
- Catching an arrow mid-flight and throwing it back (multiple high-DC checks)

## Important Limits
- **Not a "win button"** - cool actions should still have risk
- **Not repeatable** - if it works once, enemies adapt or conditions change
- **Not a replacement for actual abilities** - don't let it overshadow class features
- **DM has final say** - some things are just impossible

## Balance Considerations
- Higher DC for more impactful results
- Partial success possible (you do the cool thing but with complications)
- Critical failures can be spectacularly bad
- One-time circumstances (can't become a standard tactic)

## Player Expectations
- Describe what you want to DO, not what mechanical effect you want
- Accept that "cool" doesn't guarantee success
- Accept DM's ruling on whether it's possible
- Understand this is a privilege, not a right

---

**When in doubt:** Ask yourself "Would this make a great story?" If yes, and it's not game-breaking, allow it with appropriate checks.
